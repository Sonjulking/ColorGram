## 프로젝트 개요
### **Colorgram**은 음악의 분위기와 템포를 색깔로 설정하여  
### 해당 색상 기반으로 음악을 분류할 수 있습니다.  
### 또한 음악을 듣는 사용자간의 소통을 증진하기 위한 JavaFX 프로젝트입니다.
